# Pathfinder Studio Plugin

Build the local development plugin from the repository root:

```sh
rojo build plugin/default.project.json -o Pathfinder.rbxm
```

In Roblox Studio, choose **Plugins → Plugins Folder**, copy `Pathfinder.rbxm` into that folder, and restart Studio.

Alpha 1 includes the docked Studio interface and secure account pairing. Build execution intentionally remains disabled until the reviewed executor lands in Alpha 2.
